# Danger Game!

Danger Game is a fast-paced text-based adventure game written in Python. Choose wisely and act quickly, as the stakes increase the further you progress. Perfect for anyone who loves quick decision-making games or wants to explore Python-based gaming!

## Game Objective
The goal is to survive! Choose the only correct door within the allotted time. 
- The number of doors increases with each correct choice.
- The timer shrinks, reducing your chances of success the deeper you go.

The correct answer will always be random, so can you beat the odds and reach the **Treasure Room**? A special surprise awaits those who succeed!

## How to Run
1. Clone this repository:
   ```
   git clone https://github.com/Skorgum/danger-game.git
   ```
2. Navigate into the project's directory:
   ```
   cd danger-game
   ```
3. (Optional) Set up a virtual environment to isolate dependencies:
   ```
   python3 -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
4. Run the game:
   ```
   python main.py
   ```
