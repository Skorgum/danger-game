# Danger Game!


Simple text based adventure game in Python
Objective: Choose the only correct door. Number of doors increases with each correct choice, decreasing your odds of success the further you progress. The correct answer will always be random.
A special surprise awaits at the end. Can you beat the odds and make it to the Treasure Room?

This is a continuous work in progress

## How to run

1. Clone the repo
2. Ensure you have Python installed
3. Either type `python3 main.py` or `python main.py` depending on your Python installation
